# Navigation & Layout

## Summary
Global navigation provides quick access to all major areas and adapts to desktop and mobile layouts. The footer provides links to legal and help pages.

## User Flow
- Use the top navbar to move between Home, Stories, Create, How-To, and Profile.
- On mobile, open the burger menu to access the same links and toggles.

## Key Behaviors
- Sticky navbar with active link highlighting.
- Mobile menu expands over the page for smaller screens.
- Footer links to How-To and Imprint.
