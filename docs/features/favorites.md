# Favorites

## Summary
Stories can be saved as favorites and accessed later from the profile page.

## User Flow
- Click the favorite button on a story detail page.
- View favorites in the profile page list.

## Key Behaviors
- Favorites are synced with the backend and mirrored in localStorage.
- The favorite state is reflected in the story detail UI.
