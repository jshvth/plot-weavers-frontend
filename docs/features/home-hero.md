# Home & Hero

## Summary
The homepage introduces PlotWeavers with a hero section and shows the newest stories underneath.

## User Flow
- Land on the homepage `/` and see the hero headline, subtitle, and CTAs.
- Scroll to the "Newest Stories" section to explore recent content.

## Key Behaviors
- Hero contains primary call-to-action buttons for creating or discovering stories.
- The newest stories list shows the latest three stories fetched from the backend.
